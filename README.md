# 📊 Zoop Megastore - Análise de Dados

## 🎯 Objetivo

Analisar dados de vendas para gerar insights estratégicos sobre faturamento, comportamento de clientes e performance de produtos.

---

## 🛠️ Tecnologias utilizadas

* Python (Pandas, Matplotlib)
* MySQL
* Power BI

---

## 📁 Estrutura do Projeto

* `data/` → arquivos CSV
* `sql/` → criação de tabelas e consultas
* `python/` → tratamento e análise de dados
* `powerbi/` → dashboard interativo

---

## 📊 Principais Análises

* Faturamento total e por mês
* Top 5 produtos mais vendidos
* Top 5 clientes
* Comparação entre clientes ZOOP Prime vs não assinantes
* Faturamento por categoria e estado

---

## 📈 Dashboard (Power BI)

Inclui:

* KPI de faturamento total
* Média por cliente
* Mapa por estado
* Gráfico de tendência mensal
* Ranking de produtos

---

## 🚀 Insight Estratégico

A principal iniciativa recomendada é:

👉 Aumentar a adesão ao ZOOP Prime, focando nas categorias de maior faturamento (periféricos e livros eletrônicos), visando aumentar o ticket médio e a recorrência de compras.

---

## 📌 Como rodar o projeto

1. Importar os CSV no MySQL
2. Executar scripts SQL
3. Rodar análise em Python
4. Abrir o dashboard no Power BI
